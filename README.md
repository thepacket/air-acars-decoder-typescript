# air-acars-decoder-typescript

ACARS (Aircraft Communications Addressing and Reporting System) is a digital datalink system used in aviation for sending short, text-based messages between aircraft and ground stations via radio or satellite. It acts as a text-messaging system, automating reports (like OOOI times) and allowing pilots to receive weather updates, flight plans, and clearances, reducing voice radio traffic.

This project is a Typescript library derived from Airframes' acars-decoder-typescript for decoding ACARS messages.
